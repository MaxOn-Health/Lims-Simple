# LIMS - Laboratory Information Management System

A comprehensive LIMS platform for diagnostic centers managing health camps for employees.

## Project Structure

This repository contains two main components:

- **lims-backend/** - NestJS backend API
- **lims-frontend/** - Next.js frontend application (to be implemented)

## Documentation

- [Backend Development Guide](./BACKEND_DEVELOPMENT_GUIDE.md) - Complete backend development guide
- [Project Plan](./PROJECT_PLAN.md) - Overall project plan and architecture

## Quick Start

### Backend Setup

```bash
cd lims-backend
npm install
cp .env.example .env
# Update .env with your database credentials
npm run start:dev
```

### Frontend Setup

```bash
cd lims-frontend
# Frontend setup instructions will be added here
```
